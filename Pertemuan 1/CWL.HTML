Cerita Waktu Liburan

Saat libur kuliah, saya sering menghabiskan waktu dengan ikut bapak ke kebun. Suatu hari, ketika berada di kebun, saya membantu bapak dengan membakar pelepah sawit yang sudah menumpuk. Awalnya saya mengira itu hal biasa, tetapi ternyata api yang saya nyalakan hampir merambat ke kebun orang lain. Menyadari hal itu, bapak langsung menegur dan memarahi saya karena membakarnya sembarangan. Saat itu saya merasa takut sekaligus menyesal, karena tidak berhati-hati. Setelah selesai dari kebun, saya pulang ke rumah. Namun hari itu belum benar-benar berakhir, karena saya berencana belajar membawa mobil. Sejujurnya, saya sempat merasa ragu dan takut, sebab menurut saya belajar mengemudi mobil jauh lebih sulit dibandingkan dengan mengendarai motor. Dan ternyata, kekhawatiran saya memang terjadi.
